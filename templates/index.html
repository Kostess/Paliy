Машинное обучение

Область компьютерных наук, которая использует статистические методы и алгоритмы для автоматического обучения и улучшения систем.

Модели машинного обучения

KNN (k-nearest neighbors)

Простая модель машинного обучения, основанная на принципе поиска k ближайших соседей.

Линейная регрессия

Модель, которая строит прямую линию, наилучшим образом аппроксимирующую набор точек данных.

Логистическая регрессия

Статистический метод, который позволяет предсказать вероятность того, что событие произойдет.

Дерево решений

Модель машинного обучения, представляющая собой древовидную структуру, где каждый узел представляет собой тестовый вопрос.

TensorFlow

Распознавание одежды

Приложение, которое использует TensorFlow для распознавания различных видов одежды на изображениях.

Распознавание еды

Приложение, которое использует TensorFlow для распознавания различных видов еды на изображениях.

Распознавание частей ПК

Приложение, которое использует TensorFlow для распознавания различных частей компьютера на изображениях.

PyTorch

Детектирование

Процесс автоматического обнаружения и локализации объектов определенных классов в изображениях или видеопотоках. Это одна из ключевых задач в области компьютерного зрения и искусственного интеллекта.
