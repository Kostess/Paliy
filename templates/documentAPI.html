Описание API

API предоставляет возможность прогнозирования значения на основе входных данных, таких как рост, вес и пол человека. Результат прогноза возвращается в формате JSON.

Маршрут API

URL: http://localhost:5000/api_linear

Метод: GET

Параметры запроса

  • height (float): Рост человека в метрах.
  • weight (float): Вес человека в килограммах.
  • gender (string): Пол человека. Принимает значения: 'м' (мужской) или 'ж' (женский).

Пример использования API

Ввод данных пользователем:


height = input('Введите рост = ')
weight = input('Введите вес = ')
gender = input('Введите пол = ')
if gender == 'м':
    gender = 1
else:
    gender = 0

Отправка запроса к API:


response = get('http://localhost:5000/api_linear',
               json={'height': height,
                     'weight': weight,
                     'gender': gender})

Обработка ответа:


result = response.json()
print(result)

Возвращаемые данные

  • result (float): Прогнозируемое значение, полученное с помощью линейной модели.

Примечания

  • Входные данные должны быть корректными числами для роста и веса, а пол должен быть указан в виде строки 'м' или 'ж'.
  • Если входные данные некорректны или отсутствуют, API может вернуть ошибку.
